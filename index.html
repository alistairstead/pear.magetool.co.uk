MageTool PEAR channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover pear.magetool.co.uk
Listing available packages: 
pear remote-list -c magetool
Installing a package: 
pear install magetool/package_name
Installing a specific version/stability: 
pear install magetool/package_name-1.0.0
pear install magetool/package_name-beta
Receiving updates via a feed: 
http://pear.magetool.co.uk//feed.xml

MageTool - Additional ZF tools specifically for use during Magento development.

Additional ZF tools specifically for use during Magento development. Adding commands that will be useful during development with Magento. Command include: Clear cache, enable / disable cache and create admin user.
